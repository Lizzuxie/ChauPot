# Chau's Pot Overview 
A responsive restaurant website design, focused on creating a warm and engaging online experience for customers.
---
## Overview
This project was created for a Front-End Development I course. The goal was not just to build a visually appealing website but to understand the structure of HTML, the style of CSS, and the basics of JavaScript. 

The website includes: 
- Homepage hero section
- About Section
- Menu Section
- Responsive navigation menu for mobile devices

---
## Features
- Responsive design for desktop and mobile
- Interactive navigation menu
- Organized section-based layout

## Technologies Used
- HTML5
- CSS
- JavaScript
- Figma (for layout and design)
- Visual Code

## Design Goals
The design direction focused on:
- Warm and inviting visuals
- Mobile-first resonpsiveness
